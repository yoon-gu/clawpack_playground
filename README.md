# Installation on El Capitan 

## Fortran Compiler
1. Do Not Use ``brew install gcc``
2. Visit http://hpc.sourceforge.net and download ``gcc-x.x-bin.tar``
3. run ``sudo tar -xvf gcc-5.3-bin.tar -C /``
